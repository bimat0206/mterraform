# Changelog

All notable changes to the AWS Security Hub module will be documented in this file.

## [1.0.0] - 2024-11-07

### Added
- Initial release of AWS Security Hub module
- Security Hub account with configurable control finding generator
- Organization-level configuration with auto-enable for member accounts
- CIS AWS Foundations Benchmark (v1.2.0, v1.4.0)
- AWS Foundational Security Best Practices standard
- PCI DSS standard (optional)
- NIST SP 800-53 Rev. 5 standard (optional)
- Automatic product integrations (GuardDuty, Config, Inspector, etc.)
- Cross-region finding aggregator
- CloudWatch Events integration for real-time notifications
- SNS topic with email subscriptions for alerts
- Configurable severity and workflow status filtering
- Comprehensive outputs for monitoring and integration
