# Changelog

All notable changes to the AWS Config module will be documented in this file.

## [1.0.0] - 2024-11-07

### Added
- Initial release of AWS Config module
- Configuration recorder with continuous or daily recording
- Delivery channel with S3 bucket and SNS notifications
- Organization-level aggregator for multi-account visibility
- Support for AWS managed Config rules
- IAM role with proper permissions for Config
- S3 bucket with encryption, versioning, and lifecycle policies
- SNS topic with email subscriptions for notifications
- Comprehensive outputs for integration with other modules
