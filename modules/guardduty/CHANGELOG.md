# Changelog

All notable changes to the AWS GuardDuty module will be documented in this file.

## [1.0.0] - 2024-11-07

### Added
- Initial release of AWS GuardDuty module
- GuardDuty detector with configurable finding frequency
- S3 Protection for monitoring S3 data events
- EKS Protection with runtime monitoring and addon management
- Malware Protection for EBS volume scanning
- RDS Protection for database login monitoring
- Lambda Protection for network activity monitoring
- Organization-level configuration with auto-enable for member accounts
- S3 bucket for findings export with encryption and versioning
- CloudWatch Events integration for real-time notifications
- SNS topic with email subscriptions for alerts
- Configurable severity filtering for notifications
- Comprehensive outputs for monitoring and integration
